A MapReduce implementation of the simple probabilistic model for multi-touch attribution
described in [Shao and Li (2011)](http://users.cis.fiu.edu/~lzhen001/activities/KDD2011Program/docs/p258.pdf).

To set it up, run `mvn install`. To execute the scripts, use the `exec.sh` script in the top-level directory.
It expects `HADOOP_PREFIX` to point to your local Hadoop client.
